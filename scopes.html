2. Define Global Scope and Local Scope in JavaScript

Global scope: A variable that is defined outside of any function or block has global scope. It can be accessed from anywhere in the program, including inside functions and blocks.

Local scope: A variable that is defined inside a function or block has local scope. It can only be accessed from within the function or block where it is defined.
